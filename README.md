jLasni
=====
**Librería de gestión de mensajes de error para Java. Porque usar el depurador es de pussys.**

#### Ejemplos

  ```java
  public static void main(String[] args) {
    // Inicializamos jLasni
    Jlasni jlasni = new Jlasni();
    
    // Insertamos el mensaje de error donde mejor nos venga
    jlasni.croqueta();
      // Devolverá "Me he roto *huye haciendo la croqueta*"
  }
  ```
  
#### Métodos disponibles
* **roto()**                  - Imprime *Hola me he roto :D*
* **pechote()**               - Imprime *Me pica un pechote (????)*
* **meIgnora(String nombre)** - Imprime *<Nombre> me ignora (????)*
* **botonaco()**              - Imprime *Aquí va un BOTONACO*
* **guapisimo()**             - Imprime *Guapisimo xDDDDDDDDDDDDD*
* **felicidad()**             - Imprime un mensaje de felicidad extrema aleatorio.
* **aPastar()**               - Imprime *A PASTAR*
* **croqueta()**              - Imprime *Me he roto *huye haciendo la croqueta**
